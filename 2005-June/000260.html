I think making a how-to would be a good think for
everyone. But for now, what you can do is looking at
others plugins sources. 
Also, there is a good tutorial available at
http://xfc.xfce.org/docs/tutorial/html/chapter08.html.
(but remember you want to do a panel-plugin,not an XFC
app ;) )
Now for C or C++... I think since gtk (used by Xfce)
is done in C, using C is better. But now, if you want
to use C++, you can. (from my point of view, it
depends on what you want to do and on how big it is).
