   Hi all,

   Here's the new version of the xmms panel plugin for XFCE 4. Patrick van 
Staveren was the author of the original plugin. Later I took over and 
Mario Streiber has joined (and almost entirely rewritten the code from 
scratch).

   Last time when I submitted version 0.2.1, I didn't get any response from 
the xfce-goodies-dev team, and the the webpage wasn't updated. Do I need 
to send it to another address as well? I hope this time it gets updated, 
as the change is big. Some new features (compared to the original one by 
Staveren):

  * A (hideable) scrolling song title. You can make font size and speed 
adjustments.
  * A clickable progressbar to see and set track position.
  * Mousewheel up/down changes volume.
  * A (hideable) volume bar.

  Enjoy!


  Kemal Ilgar Eroglu
