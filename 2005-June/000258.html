   Hi all,

   Version 0.3.1 can now be downloaded from the Berlios project page:

   http://developer.berlios.de/project/showfiles.php?group_id=910

   It includes the fix for remembering the song title visibility. Also 
added an option to hide the track position progressbar. Finally one little 
change to support BMP. Actually the older versions work with BMP, too -- 
an already running BMP can be controlled by this plugin. Also, the 
configure script now checks for pango >= 1.8.0. Enjoy!
