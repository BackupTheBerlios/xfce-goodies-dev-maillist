Summary: xfce4-battery-plugin leaks memory

Details: The code seems unmaintained, judging from WebCVS, so it might have no sense to submit this, but I'll try anyway.

I have evidence, anecdotal at the moment, that the plugin leakes memory. After having my laptop on for cca. 4 days, I noticed my whole swap partition was used, which is unusual. I checked in top and the xfce4-panel was using 400MB of memory (in the VIRT column). After restart, xfce4-panel was using only 18MB. Evidently, there was a leak. I started removing plugins one by one and watched the memory usage. When I removed the xfce4-battery-plugin, the memory usage of  xfce4-panel stopped growing. It is now 18MB for another 2 days.

I can perform more specific investigation, if you let me know what you need and how to do it.
