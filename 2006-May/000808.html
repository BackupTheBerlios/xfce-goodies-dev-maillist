I think we should wait with this until Xfce 4.4 is released. We need
people to work on this and we might have more time after the release
than we have now. My ideas are:

  1. Create goodies.xfce.org, install a customized DokuWiki (similar
     to the one Benny's using for Thunar) and create pages for each

        /xfce4

     and assign rules for goodie developers here as well.
