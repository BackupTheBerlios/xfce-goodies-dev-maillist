Hello

I created a new xfce panel plugin based on the xfce4-sensors-plugin,
currently called xfce4-hwsensors-plugin. I announced this in the forum
and benny told me, this was the right place. So here I am :)

Benny said he's prefer to see the two projects merged. As I made
substantial changes to the UI and some internal parts, it probably won't
be easy to merge the projects in a way which allows the plugin to look
like the current sensors plugin, thus a merge would probably mean that
the current plugin is to be replaced by mine. That was also the reason
why I decided to create a new project instead of sending a huge patch to
the original author.

The plugin still has some small issues but it should work. The current
URL is http://tools.desire.ch/xfce4-hwsensors-plugin/ (for those who'd
like to take a look). Debian packages for sid x86 are available from the
same website.
