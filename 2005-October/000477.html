A complete rewrite of the clipman plugin, now using GPtrArray, because
it's safer and easier to handle.
This release fixes a couple of bugs in the previous version and adds
some new features: Separated clipboards (in menu), colored menu items,
add same type when clipboard was empty.

Note: you can change the menu items colors in the clipboard rc file.

You can download the 0.6 version
here<http://home.quicknet.nl/qn/prive/nickschermer/files/xfce4-clipman-plugin-0.6.tar.bz2>
.

Gr. Nick
