Hi everyone,

I'm pleased to announce the new version of Xfmedia Remote Plugin for
the panel. It's been around for a while, but only known to some irc
peeps. Now that I finally got around implementing support for the new
Xfmedia D-BUS methods, it has enough functionality to be actually a bit
useful. It also supports the new (svn) panel api.

Download at berlios:
http://developer.berlios.de/project/showfiles.php?group_id=4430

A few screenshots:
http://developer.berlios.de/screenshots/?group_id=4430

One note: There is a combination of Xfmedia and D-BUS versions, namely
Xfmedia < 0.10.0svn and D-BUS > 0.31, on which there is no playlist
menu like the one in the screenshot. This is due problems that have
been fixed, so if possible, use either the development version of
Xfmedia or the old D-BUS.

Have fun,
-Pasi
