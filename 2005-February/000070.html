The WaveLAN plugin should autohide when the WaveLAN device can not be found
(since it then must be offline) if "Autohide when offline" option is selected,
but it doesn't.

Reproducible: Always
Steps to Reproduce:
1. Select the "Autohide when offline" option in preferences.
2. rmmod <device kernel module>

Actual Results:  
Quality bar goes red and tooltip reads "No such WaveLAN device".

Expected Results:  
Autohide.
