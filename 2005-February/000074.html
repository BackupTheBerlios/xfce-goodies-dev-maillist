------- Additional Comments From benny at xfce.org  2005-02-09 10:16 UTC -------
"Offline" means: The interface is present, but not active.

The reason for this definition is simple: If the wavelan plugin would hide as
well if the interface is not present, then you wouldn't be able to change it
from the user interface. Imagine, you had a "wi0" wireless card, but now bought
a new one, which is recognized as "ath0", then you wouldn't be able to
change/remove the wavelan plugin from the panel (only by editing the
contents.xml manually), because the "wi0" device is no longer present and
therefore the wavelan plugin is hidden.
