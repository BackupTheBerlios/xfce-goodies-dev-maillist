   Hi,

   First of all, congratulations to the XFCE team for their excellent work! 
Here's my tiny contribution:

   I've made several changes to Patrick van Staveren's xmms plugin:

   * Scrolling up/down changes the volume now,
   * There's a progressbar on which you can click to set the track
   position,
   * Song title now scrolls at the top (you can hide/unhide it).

   I'm not a programmer, so don't expect a first-class work :), but it 
seems to run fine. Please check it carefully; I'll happily squash the bugs 
that you find.

   I have several other ideas like optionally having another progressbar 
for volume, maybe additional buttons for xmms file add dialog etc. These 
days I'm quite busy, but hopefully I'll be able to continue the work in 
the (not so far) future.

   Enjoy!

   Ilgar
