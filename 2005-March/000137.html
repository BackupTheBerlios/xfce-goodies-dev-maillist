Hi there.

I'm considering an almost complete rewrite of the xfce4 xkb
plugin. This will probably take some time and since I'm
quite busy at work and at school both these days it's not
clear when exactly it'll be ready. :)

I think I'll implement some new features, of which the main
will be the ability to configure Xkb extension from the
plugin. I think this feature will be of pretty much value
itself, but there are other nice consequences. One of these
would be the ability to show different text/flag image
depending on the layout variant. For example there are two
main Bulgarian layouts available (bds and phonetic) and
currently there is no way to distinguish which of them is
currently selected.

I wonder what will be the most appropriate method of
displaying the alternative layout variants... When the
plugin is configured to display the layout as a flag it
will be nice to have different flag images for each
variant. The default variant I think will use the country
flag itself, but the alternative variants could use the
flag with a small indicator that this is an alternative
layout variant. This is how it's done is Mac OS X. If I
find some time I'll post a screenshot tomorrow. I don't
know if I would break some stupid software patent owned by
Apple (or whoever else) with this.

I still cannot think of some appropriate display of the
variant when selected text indication, as most of the
layout variants are pretty long strings (e.g. the bg
variant "phonetic" mentioned earlier).

I'm open to any kind of comments, suggestions, opinions,
etc. :)

Regards,
