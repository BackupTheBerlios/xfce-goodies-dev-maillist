Hmm. Logout is performed by the session manager. I don't really want to 
add special purpose code to the panel. I don't remember if the session 
manager allows custom shutdown commands.

For the screensaver the panel uses the xflock4 script. It might be 
easier to modify that instead.

> About the clock, I think it would be good to be able to set the font, 
> the clock is displayed with. Cause the LED option is too big, and the 
> number's option is too small (I use 1280x1024 screen size).
> 

The clock could be improved a lot, but I don't think I will have the 
time to work on it. I would take a patch for this one, if someone else 
were to write it.

Other possible improvements: support for different time zones, support 
for showing the date and support for user-defined formats (like the 
date-time plugin). The dialog must remain simple though, so that will be 
a bit of a challenge ;-)

> Again about the panel, I was thinking of another option to display

Not before 4.4 I think. Middle click is not a very convenient way to 
access the menu though. I already does pop up the menu if you hold the 
left mouse button, so maybe that would be enough?

Other ideas for the launcher include putting the last-used item on the 
panel (there's even a patch for that), but I think I will leave that 
until after 4.4 to decide.
