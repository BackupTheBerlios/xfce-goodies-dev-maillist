The xfce-weather-plugin 0.4.0 stores its data
in the .xfce4 directory. All other programs 
use the new default location .config/xfce4 .
The problem is that many people delete their 
.xfce4 directory when updating to 4.2..
When the .xfce4 directory does not exist, the weather
plugin will not work (displaying a network failure error).

Please change the storage directory to .config/xfce4 .

Thanks.
Markus
