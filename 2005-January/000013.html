When you change setting of the keyboard with the command "setxkbmap", the xbk
applet do not change the icon and it freezes whole panel.

Reproducible: Always
Steps to Reproduce:
1. Install at least two kb layouts
2. Install the panel, run the console and run "setxkbmap XX" where XX is code
3. The panel freezes

Actual Results:  
You need to restart X

Expected Results:  
Changed the icon, ability to re-set the layout with the clicking on the xkb
