Greetings!

This is my first mail to this list, although I've been using Xfce for about
two years.

I really like the battery plugin and have a few suggestions that would make
it (in my opinion) even better. I'm using the xfce4-battery-plugin version
0.3.0-2 package for Debian/testing.

1. When charging from AC, it would be nice to see how long it will take to
charge it to 100%. Running 'acpitool -B' shows "Remaining capacity : 25230
mWh, 85.90%, 00:31:53", could that time estimate be shown in the battery
plugin tooltip, perhaps?

2. The ability to run commands at certain percentages of remaining battery
power is really useful. However, it would be even more useful if you could
type any command into the box instead of having to choose a program. For
instance, I would like to run 'sudo hibernate' when the battery level is
critical.
