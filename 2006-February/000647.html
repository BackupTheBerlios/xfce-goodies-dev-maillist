the panel settings dialog.

the last one I filled.

Reproducible: Always
Steps to Reproduce:
1. Add a new panel.
2. Add an instance of the showdesktop plugin to the new panel.
Actual Results:  
The panel size gets bigger than configured in the panel settings dialog.

Expected Results:  
The showdesktop plugin should respect the size configured by users.
